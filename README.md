# Express Routing Practice

A basic Node.js + Express app to practice routing, serving HTML pages, and handling a simple form.

## Features

* **Basic Routing:** Handling standard GET requests.
* **Static Files:** Serving HTML pages.
* **Form Handling:** Processing POST requests from a simple form.

## Getting Started

### Prerequisites

Ensure you have **Node.js** and **npm** installed on your machine.

### Installation & Run

1.  **Install dependencies:**
    Open your terminal in the project folder and run:
    ```bash
    npm install
    ```

2.  **Start the server:**
    Run the start script to launch the application:
    ```bash
    npm start
    ```

3.  **View the app:**
    Open your web browser and navigate to:
    [http://localhost:3000](http://localhost:3000)

## Technologies Used

* Node.js
* Express.js
